# Walmart Connect

Walmart Connect advertising platform UI prototype built with React, Vite, and the Living Design component system.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) (Vite will pick the next free port if 5173 is taken).

## Build

```bash
npm run build        # Full client + server build
npm run build:pages  # Static build for GitHub Pages preview
```

## GitHub Pages preview

The site deploys automatically on push to `main` via GitHub Actions.

Live preview: **https://metooha.github.io/walmart-connect/**
